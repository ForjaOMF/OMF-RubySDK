Class: SMS send

The http SMS send interface is a service that allows you to send SMS simply with an http post.

To make it as simple as possible we have the plugin Action SMS that adds to ActionMailer methods to send sms.

To install the gem just write in your console: 

  script/plugin install http://action-sms.googlecode.com/svn/tags/action_sms

In your rails application you must create a config/sms.yml file like this

    gateway: movistar
    login: mobile
    password: service password

Code example from ruby

If you need send SMS from your ruby script: 

    require 'movistar_gateway'
    gateway = MovistarGateway.new('login', 'password')
    gateway.send(['66666666', '777777777'], 'message')


new (login, password)

Initialize with the mobile (as login) and password service


send (destinations, message)

Send the SMS to the destinatin number

Response codes
  • 1 - Bad user.
  • 4 - Service not available.
  • 5 - Bad password.
  • 11 - Bad destination.
  • 12 - Destination is empty.
